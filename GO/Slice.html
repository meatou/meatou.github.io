# 切片

# 切片定义

语法

// 方式一：定义一个切片，让切片引用已创建好的数组。
var 数组 [长度]数据类型 = [长度]数据类型{值1, 值2, 值3...}

var 切片 = 数组[起始索引:结束索引]

// 方式二：通过 make 创建切片
var 切片 []数据类型 = make([]数据类型,长度,容量)  // 切片默认值为0

// 方式三：定义一个切片，直接就指定具体数组。
var 切片[]数据类型 = []数据类型{值1, 值2, 值3...}

切片使用

切片[索引] = 值 // 切片赋值

len(切片) // 切片长度

cap(切片) // 切片容量

# 切片遍历

# for 循环遍历

语法

for 索引 := 0; 索引 < len(切片); 索引++ {
	循环体
}

# for range 循环遍历

语法

for 索引,值 := range 切片 {
	循环体
}

注意

  1. 切片引用数组时 切片 = 数组[起始索引:结束索引] 引用的元素不包含结束索引
  2. 切片初始化时，不能越界。范围在 [0-len(数组)]，但可以动态增长。
  3. cap 是一个内置函数，用于统计切片的容量，即最大可以存放多少个元素。
  4. 切片定义完后，不能马上使用。因为本身是空的，需要引用数组或make空间提供切片使用。
  5. 切片可以继续切片

# 切片增加

语法

切片 = append(切片,值1,值2,值3...)

# 切片拷贝

语法

var 切片1[]数据类型 = []数据类型{值1, 值2, 值3...}

var 切片2[]数据类型 = make([]数据类型,长度)

copy(切片2,切片1)
