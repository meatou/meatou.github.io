# 判断 IF...ELSE

if..else if..else 用来判断一个或者多个条件，然后根据条件的结果执行不同的程序块。

示例：标准写法

// 语法：
 var 变量 = 变量值
    if 变量 > 条件1 {
        执行语句1
    } else if 变量 > 条件2 && 变量 <= 条件3 {
        执行语句2
    } else {
        执行语句3
    }
// 例子：
 var dog_age = 10
    if dog_age > 10 {
        fmt.Println("A big dog")
    } else if dog_age > 1 && dog_age <= 10 {
        fmt.Println("A small dog")
    } else {
        fmt.Println("A baby dog")
    }

示例：判断条件带括号

// 语法：
var 变量1 = 变量值1
var 变量2 = 变量值2
if (变量1 == 条件1 && 变量2 == 条件2) {
        执行语句
}
// 例子：
var dog_age = 10
var dog_sex = 'M'
if (dog_age == 10 && dog_sex == 'M') {
        fmt.Println("dog")
}

提示

Go的if..else if..else语句的判断条件一般都不需要使用()。当然如果你还是愿意写，也是对的。另外如果为了将某两个或多个条件绑定在一起判断的话，还是需要括号()的。

每个判断条件的结果要么是true要么是false，所以可以使用&&，||来连接不同的条件。使用!来对一个条件取反。

# 多分支判断 switch

switch的出现是为了解决某些情况下使用if判断语句带来的繁琐之处。

示例：

// 语法：
var 变量 := 变量值
switch 变量 {
case 条件1:
    执行语句1
case 条件2:
    执行语句2
case 条件3:
    执行语句3
default:
    默认执行语句
}
// 例子：
var score int = 69
switch score / 10 {
case 10:
case 9:
    fmt.Println("优秀")
case 8:
    fmt.Println("良好")
case 7:
    fmt.Println("一般")
case 6:
    fmt.Println("及格")
default:
    mt.Println("不及格")
}

提示

switch的判断条件可以为任何数据类型。

每个case后面跟的是一个完整的程序块，该程序块不需要{}，也不需要break结尾，因为每个case都是独立的。

可以为switch提供一个默认选项default，在上面所有的case都没有满足的情况下，默认执行default后面的语句。

# 循环 for

示例：标准写法

// 语法：
var 变量 int = 初始值
for ; 变量 <= 阀值; 变量++ {
    执行语句
}
// 例子：
var i int = 1
for ; i <= 100; i++ {
    fmt.Println(i)
}

示例：精简写法

// 语法：
for 变量 := 初始值; 变量 <= 阀值; 变量++ {
    执行语句
}
// 例子：
for i := 1; i <= 100; i++ {
    fmt.Println(i)
}

示例：while写法

// 语法：
var 变量 int = 初始值
for 变量 <= 阀值 {
    执行语句
    变量++
}
// 例子：
var i int = 1
for i <= 100 {
    fmt.Println(i)
    i++
}
