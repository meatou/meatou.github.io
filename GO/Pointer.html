所谓指针其实你可以把它想像成一个箭头，这个箭头指向（存储）一个变量的地址。

因为这个箭头本身也需要变量来存储，所以也叫做指针变量。

// 例子：
func main() {
    var x int
    var x_ptr *int
    x = 10
    x_ptr = &x
    fmt.Println(x)
    fmt.Println(x_ptr)
    fmt.Println(*x_ptr)
}

输出

10
0xc084000038
10

上面例子输出x的值，x的地址和通过指针变量输出x的值，而x_ptr就是一个指针变量。

认真理清楚这两个符号的意思。

&取一个变量的地址

*取一个指针变量所指向的地址的值
