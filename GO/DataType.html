# 概念

Go语言里面，每个变量也都是有类别的，这种类别叫做数据类型(Data Type)。 Go的数据类型有两种：一种是语言内置的数据类型，另外一种是通过语言提供的自定义数据类型方法自己定义的自定义数据类型。

# 数值型（Number）

数值型有三种，一种是整数类型，另外一种是带小数的类型(一般计算机里面叫做浮点数类型)，还有一种虚数类型。

# 整数类型

数据类型 特性 范围 备注
uint8 无符号 0～255 整形
uint16 无符号 0～65535 整形
uint32 无符号 0～4294967295 整形
uint64 无符号 0～18446744037309551615 整形
int8 有符号 -128～127 整形
int16 有符号 -32768～32767 整形
int32 有符号 -2147483648～2147483647 整形
int64 有符号 -9223372036854775808～9223372036854775807 整形

# 浮点型

数据类型 范围 备注
float32 IEEE-754 32位浮点型数
float64 IEEE-754 64位浮点型数

# 虚数类型

数据类型 范围 备注
complex64 32位实数和虚数
complex128 64位实数和虚数

# 其他数据类型

数据类型 特性 备注
byte 等同于unit8，存放字节 字节类型
rune 等同于int32，存放unicode unicode类型
uint 根据操作系统位数变化 uint32或uint64
int 根据操作系统位数变化 int32或int64
uintptr 根据操作系统位数变化 32位指针或64位指针

# 数字类型操作

操作 意义 备注
+ 加法 所有数字类型
- 减法 所有数字类型
* 乘法 所有数字类型
/ 除法 所有数字类型
% 求余 整型

# 字符串类型（String）

字符串就是一串固定长度的字符连接起来的字符序列。Go的字符串是由单个字节连接起来的。（对于汉字，通常由多个字节组成）。这就是说，传统的字符串是由字符组成的，而Go的字符串不同，是由字节组成的。这一点需要注意。

字符串的表示很简单。用(`字符串`)或者("字符串")来描述。

唯一的区别是，双引号之间的转义字符会被转义，而``号之间的转义字符保持原样不变。

# 字符串类型操作

操作 意义 备注
len(字符串) 获取字符长度
字符串[索引] 获取字符串中单个字节
字符串1+字符串2 字符串拼接

# 布尔型（Bool）

布尔型是表示真值和假值的类型。可选值为true和false。

操作 意义 备注
&& and 与
|| or 或
! not 非
